On the Subject of Me

I have been designing in various mediums for more than six years, and I've also been known to throw around a line of code or two.

I graduated from The Ohio State University with degrees in Journalism and English. I was the Editor-in-chief of the Ohio State student-run daily newspaper, The Lantern.

I built this site using HTML5 and CSS3 wherever possible and while adhering to strict Web standards and progressive enhancement.

I wholeheartedly do not believe that Web sites should look the same in every browser. Nor do I think the layout needs to look the same in every resolution.

If you stick around long enough you're bound to see many changes to the look and design of this site. I consider it my personal sandbox where I can modify and change things as I please without having to compromise for older browsers.

Check out my portfolio to see what I've done and what I'm working on.
